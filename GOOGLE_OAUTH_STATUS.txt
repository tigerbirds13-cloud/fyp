🚀 HOW TO ENABLE GOOGLE OAUTH:

Step 1: Get Credentials (5 minutes)
  → Go to https://console.cloud.google.com/
  → Create Project → Enable Google+ API
  → Create OAuth 2.0 Web App credentials
  → Copy Client ID and Client Secret

Step 2: Update .env Files
  → backend/.env: Add GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET
  → frontend/.env: Add REACT_APP_GOOGLE_CLIENT_ID

Step 3: Restart Servers
  → npm start
  → Google OAuth now fully activated!
